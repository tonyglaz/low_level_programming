1.1 Постановка задачи
Написать программу, вычисляющую на ассемблере числа Стирлинга второго рода.

1.2 Неформальное описание алгоритма
В комбинаторике числом Стирлинга второго рода из n по k, обозначаемым S(n,k)
 называется количество неупорядоченных разбиений n-элементного множества на k непустых подмножеств.
S(n,k)=S(n-1,k-1)+k×S(n-1,k)
Если задано множество из n элементов, которое необходимо разбить на k непустых частей, то последний
 элемент исходного множества можно либо поместить в отдельную часть S(n-1,k-1) способами, либо поместить
 его в некоторое подмножество k×S(n-1,k) способами, поскольку каждый из S(n-1,k) способов распределения 
первых n-1 элементов по k непустым частям дает k подмножеств, с которыми можно объединить последний элемент.
